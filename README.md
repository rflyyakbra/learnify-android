Learnify adalah aplikasi pembelajaran interaktif berbasis Android yang dirancang untuk membantu siswa dalam mempersiapkan Ujian Tulis Berbasis Komputer (UTBK). Aplikasi ini menyediakan materi pembelajaran, latihan soal, try out, serta fitur pencatatan progres belajar yang dapat digunakan secara mandiri.

- **Login & Register**

  - TOBK (30 soal) dengan batas waktu
- **Achievement & Progress**
  - Target skor TOBK
  - Riwayat nilai
  - Badge/penghargaan belajar
- **Profile**
  - Ubah nama pengguna
  - Logout
